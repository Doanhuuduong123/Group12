Trò chơi giúp người dùng giải trí thông qua việc điều khiển chú rắn ăn mồi để tăng điểm số, đồng thời yêu cầu sự khéo léo để tránh va phải tường hoặc thân mình. Ứng dụng tập trung vào việc áp dụng các kiến thức về lập trình hướng đối tượng (OOP) và cấu trúc dữ liệu giải thuật.✨ Các chức năng chính (Features)[x] Điều khiển mượt mà: Sử dụng phím mũi tên để điều hướng rắn di chuyển.[x] Logic va chạm: Xử lý va chạm chính xác với tường, mồi và bản thân rắn bằng Rectangle.intersects().[x] Độ khó tăng dần: Tốc độ di chuyển của rắn tăng lên sau mỗi 5 lần ăn mồi thành công.[x] Lưu trữ kỷ lục: Hệ thống lưu và hiển thị điểm cao nhất (Highscore) thông qua File I/O.[x] Giao diện trực quan: Sử dụng thư viện Java Swing và hình ảnh (Assets) để tạo cảm giác sinh động.[x] Âm thanh & Hiệu ứng: (Tính năng mở rộng) Âm thanh khi ăn mồi và khi kết thúc trò chơi.💻 Công nghệ & Thư viện sử dụng (Technologies)Ngôn ngữ: Java (JDK 17+)Giao diện: Java Swing, AWTCơ sở dữ liệu / Lưu trữ: File Binary / Text (.dat / .txt) để lưu điểm cao.Công cụ khác: Git, GitHub, IntelliJ IDEA / VS Code.📂 Cấu trúc thư mục (Project Structure)Mã nguồn được tổ chức chặt chẽ theo mô hình MVC (Model - View - Controller):Plaintext  📦 src
   ┣ 📂 model       # Chứa các lớp thực thể: Snake, Food, Point, Board...
   ┣ 📂 view        # Chứa các lớp giao diện: GameFrame, GamePanel, MenuPanel...
   ┣ 📂 controller  # Chứa logic: GameLoop, InputHandler, CollisionController...
   ┣ 📂 utils       # Chứa các lớp tiện ích: FileHandler (lưu điểm), AssetLoader...
   ┗ 📜 Main.java   # File khởi chạy ứng dụng
